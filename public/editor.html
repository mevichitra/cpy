Terminal Access

You can interact with this note directly from your terminal. Use these commands to retrieve, pipe, or upload files:

Retrieve Text (GET)
curl http://localhost:3000/slug
Update Text (POST)
curl -d "New text" http://localhost:3000/slug
Pipe Output to Note
echo "My content" | curl -d @- http://localhost:3000/slug
Upload File Contents
curl --data-binary @file.txt http://localhost:3000/slug
